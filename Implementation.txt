  SUBPHASE 2.3 â€” Window Manager Store (Zustand) â€” CRITICAL
  ---------------------------------------------------------------------------
  stores/useWindowStore.ts:
    State:
      - windows: Map<WindowId, WindowState>
      - activeWindowId: WindowId | null
      - windowOrder: WindowId[] (untuk z-index stack)
      - nextZIndex: number

    Actions:
      - openWindow(appId): Buat WindowState baru, set sebagai focused,
        assign z-index tertinggi. Jika app sudah terbuka, focus ke window
        yang ada (bukan buka duplikat).
      - closeWindow(windowId): Hapus dari state, update focus ke window
        berikutnya di stack.
      - minimizeWindow(windowId): Set isMinimized = true, update focus
        ke window berikutnya.
      - maximizeWindow(windowId): Simpan posisi/size sekarang ke
        previousPosition/previousSize, set ke fullscreen dimension.
      - restoreWindow(windowId): Kembalikan ke previousPosition/previousSize,
        set isMaximized = false.
      - focusWindow(windowId): Set sebagai activeWindowId, ambil z-index
        tertinggi baru, pindahkan ke atas stack.
      - moveWindow(windowId, position): Update posisi {x, y}.
      - resizeWindow(windowId, size): Update ukuran {w, h}.
      - snapWindow(windowId, snapPosition): Snap ke left-half, right-half,
        atau full. Simpan posisi sebelumnya untuk restore.
      - unminimizeWindow(windowId): Set isMinimized = false, focus window.

    Computed:
      - getVisibleWindows(): Windows yang tidak minimized
      - getMinimizedWindows(): Windows yang minimized (untuk dock indicator)
      - getFocusedWindow(): Window yang aktif saat ini
      - getWindowsByZIndex(): Sorted berdasarkan z-index

    Persist:
      - Zustand persist middleware ke localStorage sebagai fallback cepat

  SUBPHASE 2.4 â€” Desktop Store (Zustand)

  SUBPHASE 7.4 â€” Animation Fine-Tuning & Performance
  ---------------------------------------------------------------------------
  lib/animations.ts:
    Shared Framer Motion variants:

    windowOpen: {
      initial: { scale: 0.85, opacity: 0 },
      animate: { scale: 1, opacity: 1 },
      transition: { type: 'spring', stiffness: 300, damping: 25 }
    }

    windowClose: {
      exit: { scale: 0.85, opacity: 0 },
      transition: { type: 'spring', stiffness: 400, damping: 30 }
    }

    dockMagnify: {
      scale calculation berdasarkan distance dari mouse position
      using useMotionValue + useTransform
    }

    fadeIn: { initial: { opacity: 0 }, animate: { opacity: 1 } }
    slideUp: { initial: { y: 20, opacity: 0 }, animate: { y: 0, opacity: 1 } }

  Performance rules:
    - will-change: transform pada dragged elements
    - GPU acceleration via transform3d(0,0,0)
    - No layout-triggering animations (avoid animating width/height directly)
    - Use transform: scale() instead
    - Lazy load app content yang belum pernah dibuka
    - RequestAnimationFrame untuk custom animations
    - Target: consistent 60fps
